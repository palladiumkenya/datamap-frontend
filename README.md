## Getting Started

1. Clone from Github


2. Install packages

```
npm install
```

3. Run project

```
npm start
```

## Technology stack

- [Material UI V5](https://mui.com/core/)
- Built with React Hooks API.
- Redux & React context API for state management.
- Redux toolkit.
- React Router for navigation routing.
- Support for react-script.
- Code splitting.
- CSS-in-JS.
